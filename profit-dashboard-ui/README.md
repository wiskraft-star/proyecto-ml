# Profit Dashboard UI (Mock)

Este ZIP contiene **solo la página UI** (App Router) para que la pegues en tu proyecto actual y la subas a GitHub.

## Ruta incluida
- `app/profit/page.tsx`

## Requisitos (ya los tenés o se instalan)
- Next.js (App Router)
- Tailwind CSS
- shadcn/ui components en `@/components/ui/*`
- `lucide-react`
- `recharts`

## Instalación rápida (si faltan deps)
```bash
npm i recharts lucide-react
```

## Cómo usarla
1. Copiá la carpeta `app/profit` dentro de tu repo (o ajustá la ruta a la que estés usando).
2. Corré el proyecto:
```bash
npm run dev
```
3. Abrí:
- `http://localhost:3000/profit`

## Nota
La página es 100% **mock UI**: no hay conexiones a ML/MP/DB todavía (eso va en etapa 2).
